### Описание
Универсальное Streamlit-приложение для экспресс-анализа табличных данных в формате CSV(Поддерживает разделитель ","). 
Автоматически определяет кодировку файла, распознаёт столбцы с датами и числовыми значениями. 
Позволяет мгновенно получить описательную статистику (среднее, медиана, СКО), построить точечные и линейные графики зависимостей, а также скачать визуализации в формате PNG. 
В основе лежит библиотека `pandas` для парсинга/агрегации и `matplotlib`/нативные компоненты Streamlit для визуализации.

### Установка зависимостей
1. Убедитесь, что установлен Python 3.9+
2. Создайте виртуальное окружение и активируйте его:
   ```bash
   python -m venv .venv
   source .venv/bin/activate   # Linux/macOS
   .venv\Scripts\activate      # Windows

Установите зависимости:
pip install -r requirements.txt

### Запуск
В терминале из папки проекта выполните:
streamlit run HW31_Main.py

Приложение откроется в браузере по адресу http://localhost:8501
