# Кредитный калькулятор на Streamlit

Интерактивное веб-приложение для расчета параметров кредита и формирования детального графика платежей. Поддерживает основные банковские методики начисления процентов.

## Функционал

- **Два типа платежей:** расчет аннуитетных (равных) и дифференцированных (уменьшающихся) платежей.
- **Динамический расчет:** мгновенное обновление данных при изменении суммы, ставки или срока.
- **Детальный график:** генерация таблицы с разбивкой каждого платежа на проценты и основной долг.
- **Учет дат:** расчет конкретных дат выплат начиная с выбранного календарного дня.
- **Итоги:** наглядный вывод общей суммы выплат и суммы переплаты по кредиту.

## Библиотеки

- [Streamlit](https://streamlit.io) — для интерфейса.
- [Pandas](https://pydata.org) — для работы с таблицами.
- [python-dateutil](https://readthedocs.io) — для корректной работы с календарными месяцами.

## Установка и запуск

1. **Установите зависимости:**
   ```bash
   .venv\Scripts\activate
   ```

2. **Запустите приложение:**
   ```bash
   streamlit run HW30_main.py
   ```
   Приложение откроется в браузере по адресу http://localhost:8501

## Как пользоваться

1. Введите **сумму кредита** и **годовую процентную ставку** в боковой панели.
2. Укажите **срок** кредитования в месяцах.
3. Выберите **тип платежа**.
4. Установите **дату первого платежа** для формирования календаря выплат.
5. Изучите итоговые показатели и разверните таблицу с графиком для детального анализа.

## Особенности реализации
- При расчетах используется `st.stop()` для предотвращения ошибок при вводе нулевых значений.
- Использован `st.expander` для компактного отображения больших таблиц.
- Финальный платеж в графике автоматически корректируется для обнуления остатка с учетом погрешностей округления.
